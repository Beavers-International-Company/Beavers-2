# Smart Scooter Campus

Smart Scooter Campus is a simple educational MVP web project for the course **IT Project Management**. It demonstrates a campus e-scooter rental service where students can register with a student email, view mock scooters, simulate a QR scan to start a ride, finish the ride, and receive eco bonuses.

## Main Features

- Student email registration with student-domain validation.
- Student profile with verified status.
- Mock campus scooter map/list.
- Scooter availability, location, battery level, and status display.
- Filter to show only available scooters.
- Simulated QR-code scan to start a rental.
- Active ride panel with scooter, start time, and status.
- Finish ride action with completed status.
- Eco bonus statistics, CO₂ saved estimate, ride count, and user rating.

## Technologies

- HTML5
- CSS3
- JavaScript
- Browser `localStorage`
- Mock data only

## How to Run the Project

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. Register with a valid student email such as `name@student.edu` or `name@kpnu.edu.ua`.
4. View scooters, start a simulated ride, finish it, and check eco bonuses.

No backend, database, APIs, payment systems, real GPS, Google Maps, or real QR scanner are required.

## Sprint 1 Description

Sprint 1 focuses on the MVP features required to show the main service flow:

1. **SCRUM-10 — Student email registration — 5 SP**  
   Validate a student email domain and create a verified student profile.
2. **SCRUM-15 — View available scooters on the map — 8 SP**  
   Display mock scooters on a simple campus map/list and filter available scooters.
3. **SCRUM-16 — Display scooter battery level — 3 SP**  
   Show battery percentages and low-battery status for scooters.
4. **SCRUM-20 — QR-code scanning simulation to start rental — 8 SP**  
   Simulate QR scanning by selecting a scooter and starting a rental.
